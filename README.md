# Maca Releases

Public release channel for the **Maca** macOS desktop app.

This repository contains **only release artifacts and the changelog** — never
the application source code.

- **Download the latest version:** https://github.com/crafter-station/maca-releases/releases/latest/download/Maca-arm64.dmg
- **All releases:** https://github.com/crafter-station/maca-releases/releases
- **Changelog:** [CHANGELOG.md](./CHANGELOG.md)

Installed apps auto-update from the GitHub Releases in this repo (via
electron-updater). Learn more at https://maca.sh
